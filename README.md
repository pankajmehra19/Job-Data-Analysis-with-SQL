Overview

This project analyzes a dataset of job postings to understand trends in the Data Analyst job market. The analysis focuses on remote opportunities, salary trends, and the technical skills most frequently required by employers. SQL is used to explore relationships between job postings, skill requirements, and compensation to generate insights about current market demand.

Objectives

Identify high-paying Data Analyst roles and analyze salary trends.

Determine the most in-demand skills required for Data Analyst positions.

Examine the relationship between technical skills and average salary levels.

Analyze remote job opportunities within the Data Analyst job market.

Key Analyses Performed

Identification of the highest paying Data Analyst roles based on salary data.

Analysis of the most in-demand technical skills across job postings.

How to Use This Project

Load the dataset into a SQL database using the scripts provided in the sql_load directory.

Execute the SQL queries located in the Project_sql folder.

Review the outputs to explore trends related to salary, job demand, and required skills.

Dataset

The dataset used in this project contains information about job postings, including job titles, salary data, location, remote work availability, and required technical skills. The dataset is commonly used for SQL-based data analysis practice.
